## ZIRIZ Client & Contract

Basic command for this project
```
yarn install // to install dependencies

yarn contract:format && yarn contract:clippy // run formatter and clippy

yarn contract:test // to run a unit test

yarn contract:build // to build contract

yarn contract:deploy-testnet // deploy contract to testnet

yarn contract:link // binding contract client to ts

yarn contract:add // add contract to to ts project

yarn init // initialize contract

yarn test // run integration test on testnet
```
